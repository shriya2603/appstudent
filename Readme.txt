Setup Details 

    step 1: To install all the modules/packages used in project 
        Type "npm install" in command prompt.

    step 2: setup the mongoDB database 
        Here, I have used  "apiAuth" as a mongoDb database name. Every time u want to start your project u have to start MongoDb database server.

    step 3: To start the server 
        "node app" in command prompt.

Code Details 
    1. Before starting the project u have to change the google clientID and clientPassword in config/index.js.
